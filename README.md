**1.Airbnb Global Performance insights dashboard**

A dynamic, interactive data visualisation tool built to explore Airbnb performance data for over 250k listings in 10 major cities, including information about hosts, pricing, location, and room type, along with over 5 million historical reviews.

**2.Description**

The Airbnb Global Performance Insights Dashboard provides a centralized, data-driven view of how listings perform across different regions, focusing on both listing metrics and guest reviews.It enables users to monitor key indicators such as occupancy rates, pricing trends, and booking performance over time.The dashboard offers a comprehensive global view that combines operational performance with customer sentiment, supporting strategic decisions to optimize listings, enhance guest experience, and maximize revenue.
